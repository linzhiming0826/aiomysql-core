Aiomysql-Core
=============

.. module:: aiomysql_core

**Aiomysql-Core** is an extension for aiomysql that adds support for quickly
building APIs.  It is a lightweight abstraction that works with your
existing libraries.  Aiomysql-Core encourages best practices with minimal
setup. If you are familiar with aiomysql, Aiomysql-Core should be easy to pick up.
