Installation
============

.. currentmodule:: aiomysql_core

Install Aiomysql-Core with ``pip`` ::

    pip install aiomysql-core


The development version can be downloaded from `its page at GitHub
<https://github.com/linzhiming0826/aiomysql-core>`_. ::

    git clone https://github.com/linzhiming0826/aiomysql-core.git
    cd aiomysql-core
    python setup.py develop


Aiomysql-Core has the following dependencies (which will be automatically
installed if you use ``pip``):


* `aiomysql <https://github.com/aio-libs/aiomysql>`_ version 0.0.20 or greater

Aiomysql-Core requires Python version 3.6 or greater
